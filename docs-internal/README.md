# Internal Documentation

Internal project documentation, guides, and plans for the SalesOS team.

## Structure

| Directory | Contents |
|-----------|----------|
| `guides/` | Admin UI, navigation, setup, and user role guides |
| `plans/` | Feature roadmaps, AI builder plans, keyword strategy |
| `seo/` | SEO checklists, guides, and implementation summaries |
| `security/` | Data import security docs and quick references |
| `infrastructure/` | AWS ALB/SSL setup instructions and CLI commands |
| `migration/` | CSV migration fixes and production migration docs |

> **Note**: The public-facing documentation site lives in `/docs/` (a separate Next.js app).
