Q: I don't see "CRM Migration" in my Settings menu
A: You need ADMIN or OWNER role. Contact your organization owner.

Q: I get "403 Forbidden" when accessing migration
A: Backend RBAC enforces ADMIN/OWNER only. Request role upgrade.

Q: My CSV upload fails
A: Check file size (<50MB), format (CSV/Excel), and encoding (UTF-8).

Q: AI field mapping is incorrect
A: Use manual override dropdowns to correct mappings before import.

Q: Import failed with errors
A: Download error log, fix data issues, and re-import failed records.

Q: Can I undo a migration?
A: No automatic rollback. Use Migration History to track what was imported.

Q: How do I handle custom fields?
A: Map custom CSV columns to custom SalesOS fields in step 3.
